## XScanner
Full Network Scanner Project (Worker)

## 全网web扫描系统

该项目主要为全网扫描web服务建立。收集网络信息。

##环境

python2.7

redis

elasticsearch


##安装

pip install -r requirements.txt

下载并启动ielasticsearch

下载并启动redis


##运行

首先启动redis、elasticsearch

windows系统

startWork.bat  			多任务

startWork_single.bat   单任务

linux系统

startWork.sh  			多任务

startWork_single.sh   单任务
